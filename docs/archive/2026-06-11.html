🌍 Svět

Ivanti Sentry zranitelnost aktivně zneužívána — Kritická zranitelnost v Ivanti Sentry (maximální závažnosti) je nyní aktivně zneužívána útočníky. Ti mohou získat root oprávnění a spouštět libovolný kód na exponovaných zařízeních. Organizace používající Ivanti Sentry by měly neprodleně aplikovat dostupné záplaty. [Zdroj: BleepingComputer] URL: https://www.bleepingcomputer.com/news/security/max-severity-ivanti-sentry-vulnerability-now-exploited-in-attacks/

Zranitelnost Langflow platformy pro AI zneužívána — Závažná zranitelnost typu path traversal (CVE-2026-5027) v platformě Langflow pro vývoj AI aplikací je aktivně zneužívána. Útočníci ji využívají k zápisu libovolných souborů na exponované servery. Uživatelé Langflow by měli okamžitě aktualizovat své instalace. [Zdroj: BleepingComputer] URL: https://www.bleepingcomputer.com/news/security/path-traversal-flaw-in-ai-dev-platform-langflow-exploited-in-attacks/

Microsoft záplatuje zero-day v Exchange Serveru — Microsoft vydal záplatu pro aktivně zneužívanou zero-day zranitelnost v Exchange Serveru, která útočníkům umožňuje spouštět libovolný JavaScript kód prostřednictvím XSS útoků cílených na Outlook Web Access. Je klíčové co nejdříve aplikovat tuto aktualizaci. [Zdroj: BleepingComputer] URL: https://www.bleepingcomputer.com/news/microsoft/microsoft-patches-exchange-server-zero-day-exploited-in-attacks/

GitHub vypne npm install skripty ve výchozím nastavení — GitHub oznámil zásadní změny v nadcházející verzi npm v12, které ve výchozím nastavení vypnou spouštění instalačních skriptů. Cílem je výrazně snížit riziko útoků na dodavatelský řetězec softwaru. Vývojáři by se měli připravit na tyto změny. [Zdroj: The Hacker News] URL: https://thehackernews.com/2026/06/github-to-disable-npm-install-scripts.html

Servery Oracle PeopleSoft napadeny gangem ShinyHunters — Extortionistický gang ShinyHunters cílí na servery Oracle PeopleSoft a tvrdí, že ukradl data z více než 100 organizací. Tento incident podtrhuje důležitost robustní ochrany dat a včasného patchování podnikových systémů. [Zdroj: BleepingComputer] URL: https://www.bleepingcomputer.com/news/security/oracle-peoplesoft-servers-hacked-in-shinyhunters-data-theft-attacks/

⚠️ Příběh z praxe

Organizace spoléhající na Ivanti Sentry zařízení se stala terčem útoku, protože neaplikovala včas kritickou bezpečnostní záplatu. Útočníci využili známou zranitelnost k získání root oprávnění a spuštění libovolného kódu na exponovaném zařízení, čímž získali plnou kontrolu nad systémem. Oběť zanedbala pravidelné monitorování dostupnosti záplat a jejich okamžitou implementaci. Měla mít zavedený proces pro rychlou reakci na oznámení o kritických zranitelnostech a automatizované nasazování aktualizací.

🛡️ Praktický tip

Co udělat: Okamžitě zkontrolujte a aplikujte všechny dostupné bezpečnostní záplaty pro vaše systémy, zejména pro Ivanti Sentry, Microsoft Exchange a platformu Langflow.

Proč: Existuje vysoké riziko aktivního zneužívání těchto zranitelností, které může vést k úplné kompromitaci vašich systémů a úniku dat.

Jak: Pravidelně sledujte bezpečnostní bulletiny výrobců softwaru, používejte nástroje pro správu záplat a prioritizujte nasazení kritických aktualizací.
