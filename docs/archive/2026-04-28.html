🌍 Svět

Microsoft potvrdil aktivní zneužívání zranitelnosti ve Windows Shell — Microsoft revidoval své doporučení k již opravené kritické zranitelnosti CVE-2026-32202 ve Windows Shell a potvrdil, že je aktivně zneužívána. Tato chyba umožňuje útočníkům eskalaci oprávnění. Je klíčové zajistit, aby byly všechny systémy Windows co nejdříve aktualizovány. [Zdroj: The Hacker News] URL: https://thehackernews.com/2026/04/microsoft-confirms-active-exploitation.html

Opravena kritická chyba v Microsoft Entra ID — Microsoft opravil zranitelnost v Entra ID (dříve Azure AD), která se týkala administrativní role určené pro AI agenty. Tato chyba mohla umožnit eskalaci oprávnění a převzetí identity. Organizace by měly zkontrolovat konfigurace a oprávnění v Entra ID. [Zdroj: The Hacker News] URL: https://thehackernews.com/2026/04/microsoft-patches-entra-id-role-flaw.html

Malware GlassWorm se vrací přes rozšíření OpenVSX — Nová vlna kampaně GlassWorm zneužívá ekosystém OpenVSX, kde bylo objeveno 73 "spících" rozšíření, která se po aktualizaci stávají škodlivými. Tato rozšíření jsou navržena k odcizení informací. Vývojáři a organizace používající VS Code by měli být extrémně opatrní při instalaci rozšíření a pravidelně je kontrolovat. [Zdroj: BleepingComputer] URL: https://www.bleepingcomputer.com/news/security/glassworm-malware-attacks-return-via-73-openvsx-sleeper-extensions/

Populární PyPI balíček napaden, šíří infostealer — Útočníci kompromitovali populární balíček `elementary-data` v Python Package Index (PyPI), který má 1,1 milionu stažení měsíčně, a nahráli jeho škodlivou verzi. Ta slouží k odcizení citlivých dat vývojářů a kryptoměnových peněženek. Je to další připomínka rizik v dodavatelských řetězcích softwaru. [Zdroj: BleepingComputer] URL: https://www.bleepingcomputer.com/news/security/pypi-package-with-11m-monthly-downloads-hacked-to-push-infostealer/

Čínský hacker napojený na Silk Typhoon vydán do USA — Čínský občan obviněný z členství v hackerské skupině Silk Typhoon, která prováděla kyberšpionáž pro čínské zpravodajské služby, byl vydán z Itálie do Spojených států. Je obviněn z kybernetických útoků zaměřených na výzkum COVID-19. Tato zpráva podtrhuje pokračující úsilí v boji proti státem sponzorovaným kybernetickým hrozbám. [Zdroj: The Hacker News] URL: https://thehackernews.com/2026/04/chinese-silk-typhoon-hacker-extradited.html

⚠️ Příběh z praxe

Online obchodní platforma Robinhood čelila zneužití svého procesu vytváření účtů, kdy útočníci vkládali phishingové zprávy do legitimních e-mailů. Oběti obdržely zdánlivě oficiální upozornění na vytvoření účtu, která je naváděla k zadání přihlašovacích údajů na podvodné stránce. Zanedbána byla detailní kontrola URL adresy a ověření nečekané zprávy. Uživatelé by si měli vždy ověřit legitimitu zprávy mimo e-mail, například přímým přihlášením na platformu přes známou URL.

🛡️ Praktický tip

Co udělat: Okamžitě aplikujte bezpečnostní aktualizace pro Windows a zkontrolujte konfiguraci Entra ID.

Proč: Microsoft potvrdil aktivní zneužívání kritické zranitelnosti ve Windows Shell a zároveň opravil chybu v Entra ID umožňující eskalaci oprávnění. Tyto hrozby představují vysoké riziko pro integritu systémů a identit.

Jak: Zajistěte, aby všechny systémy Windows měly nainstalované nejnovější bezpečnostní záplaty. Pro Entra ID prověřte role a oprávnění, zejména ty spojené s AI agenty, a aplikujte doporučené konfigurace od Microsoftu pro posílení zabezpečení identit.
